Como posso ajudar?

Há muitas maneiras de ajudar se estiver interessado em ver a web crescer e ser ainda mais útil...

Coloque alguns dados
    Há muitas maneiras de fazer isso. A web precisa de dados brutos -- hipertexto fresco ou arquivos de texto simples antigos, ou servidores inteligentes que dão visualizações de bancos de dados existentes. Veja mais detalhes, etiqueta.
Sugira que outra pessoa faça
    Talvez você conheça um sistema que seria legal ter na web. Que tal sugerir à pessoa envolvida que eles colocaram um servidor W3?
Gerenciar uma área de assunto
    Se você sabe algo do que está acontecendo em um determinado campo, organização ou país, gostaria de manter atualizada uma visão geral dos dados on-line?
Escreva algum software
    Temos uma grande lista de coisas a serem feitas. Ajude-se - todas as contribuições recebidas com gately! veja a lista .
Envie-nos sugestões
    Adoramos receber correspondência... www-bug@info.cern.ch
Diga aos seus amigos
    Instale/instale o software do cliente em seu site. Cite as coisas pelo endereço W3 para permitir que os usuários do W3 as peguem diretamente.
