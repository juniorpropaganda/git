Ajuda

Qual programa você está usando para o qual você quer ajuda on-line?

Navegador de modo de linha
    Usando o navegador de modo de linha, selecione uma opção com um número e pressione RETURN
Viola
    Ajuda fpr na Viola, basta pressionar o ícone de ponto de interrogação (?) perto do canto superior direito.
NeXTStep
    Para selecionar ajuda com este navegador, use Ajuda no menu Informações ou selecione esta opção sagacidade do mouse (clique duas vezes).
Midaswww
    Para obter ajuda neste navegador, selecione esta opção com o mouse.

Veja também o manual do usuário do servidor W3. Para outras perguntas, navegue na página do projeto W3.
