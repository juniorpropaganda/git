WorldWideWeb - Resumo

O projeto WWW mescla as técnicas de recuperação de informações e hipertexto para tornar um sistema de informações global fácil, mas poderoso.

O projeto baseia-se na filosofia de que muitas informações acadêmicas devem estar livremente disponíveis para qualquer pessoa. O objetivo é permitir o compartilhamento de informações em equipes dispersas internacionalmente e a disseminação de informações por grupos de apoio. Originalmente voltada para a comunidade de Física de Alta Energia, ela se espalhou para outras áreas e atraiu muito interesse em áreas de suporte ao usuário, descoberta de recursos e áreas de trabalho colaborativas.

Visão do leitor


O mundo WWW consiste em documentos e links. Índices são documentos especiais que, em vez de serem lidos, podem ser pesquisados. O resultado dessa pesquisa é outro documento ("virtual") contendo links para os documentos encontrados. Um protocolo simples (" HTTP ") é usado para permitir que um programa de navegador solicite uma pesquisa de palavras-chave por um servidor de informações remotas.

A web contém documentos em muitos formatos. Esses documentos que são hipertexto, (reais ou virtuais) contêm links para outros documentos, ou lugares dentro de documentos. Todos os documentos, sejam reais, virtuais ou índices, são semelhantes ao leitor e estão contidos no mesmo esquema de endereçamento.

Para seguir um link, um leitor clica com um mouse (ou digita em um número se ele ou ela não tiver mouse). Para pesquisar e indexar, um leitor dá palavras-chave (ou outros critérios de pesquisa). Estas são as únicas operações necessárias para acessar todo o mundo dos dados.

Visualização do provedor de informações


Os navegadores WWW podem acessar muitos sistemas de dados existentes através de protocolos existentes (FTP, NNTP) ou via HTTP e um gateway. Dessa forma, a massa crítica de dados é rapidamente ultrapassada, e o uso crescente do sistema por leitores e fornecedores de informações incentivam uns aos outros.

Fazer uma web é tão simples quanto escrever alguns arquivos SGML que apontam para seus dados existentes. Torná-lo público envolve executar o FTP ou HTTP daemon , e fazer pelo menos um link em sua web a partir de outro. Na verdade, qualquer arquivo disponível por FTP anônimo pode ser imediatamente vinculado em uma web. O pequeno esforço inicial foi projetado para permitir pequenas contribuições. Na outra ponta da escala, grandes provedores de informações podem fornecer um servidor HTTP com indexação completa de texto ou palavras-chave. Isso pode permitir o acesso a um grande banco de dados existente sem alterar a forma como o banco de dados é gerenciado. Esses gateways já foram feitos nos sistemas VMS/Help da Digital, Universidade Técnica do "Hiper-G" da Graz e sistemas "W.A.I.S." da Thinking Machine.

O modelo WWW supera as frustrantes incompatibilidades do formato de dados entre fornecedores e leitores, permitindo a negociação de formato entre um navegador inteligente e um servidor inteligente. Isso deve fornecer uma base para a extensão na multimídia e permitir que aqueles que compartilham padrões de aplicativos façam uso total deles através da web.

Este resumo não descreve as muitas possibilidades interessantes abertas pelo projeto WWW, como o cache eficiente de documentos. a redução de cópias desatualizadas redundantes, e o uso de daemons de conhecimento. Há mais informações na documentação do projeto online, incluindo alguns antecedentes sobre hipertexto e muitas notas técnicas.

Experimente.


Você pode experimentar o navegador do modo de linha simples por rede de info.cern.ch (sem usuário ou senha. Do Reino Unido JANET, use o Gateway). Você também pode saber mais sobre WWW desta forma. Este é o navegador menos sofisticado - lembre-se que os orientados para janelas são muito mais inteligentes!

É muito mais eficiente instalar um navegador em sua própria máquina. O navegador de modo de linha está atualmente disponível em formulário de origem por FTP anônimo do nó info.cern.ch [atualmente 128.141.201.74] como

(v.vv é o número da versão - pegue o mais recente.)
